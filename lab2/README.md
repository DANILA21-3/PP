###Инструкция к запуску

Для запуска программы необходимо запустить `run.bat`. После его запуска ввести: 
1)Размер перемножаемых матриц; 
2)Количество используемых потоков. 
После выполнения вычислений Будет получено затраченное время и результат верификации. Файлы с исходными и итоговый матрицей будут хранится в текущей директории.
