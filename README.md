# Лабораторная работа №1

В ходе лабораторной работы были реализованы следующие компоненты:

1. **matrix_generation.py** - создание матриц с хаотичным заполнением целых значений от 0 до 10
2. **main.cpp** - перемножение матриц с использованием собственного алгоритма
3. **verification.py** - верификация полученного результата
4. **reporting** - введение отчётности по полученным результатам

## Технологии

- **Python**: NumPy для взаимодействия с матрицами и верификации
- **C++**: chrono для замера времени выполнения

## Инструкция к запуску

Для запуска программы необходимо запустить 'run.bat'. После его запуска ввести размер перемножаемых матриц. Будет получено время вычисления и результат верификации. Файл с исходными и итоговый матрицей будет хранится в текущей директории.
